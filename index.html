My Story

I'm Bandaru Sai Sumanth, a passionate web developer and programmer who loves turning ideas into real, working products. My journey began with curiosity about how websites and software are built — and that curiosity quickly grew into a full-on passion.

I started with the fundamentals — HTML, CSS, and JavaScript — and built my Personal Portfolio to showcase my work. From there I pushed myself further, creating a Resume Builder Pro that lets users generate professional resumes with a live preview, and WeatherSphere Pro, a real-time weather app powered by live API data.

On the backend side, I built a Python Scientific Calculator with History, which deepened my understanding of logic, math operations, and clean code structure. Each project taught me something new and pushed me to grow as a developer.

I believe in learning by building. Every line of code I write brings me closer to creating products that are not just functional, but genuinely useful and beautiful.
